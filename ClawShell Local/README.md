# ClawShell Local

端侧安装单元，运行在 Mac/车机等本地设备上。

## 安装

```bash
cd ClawShell Local
chmod +x install.sh scripts/*.sh
./install.sh
```

## 功能

- 自动发现 Hermes / OpenClaw / 悟空
- 云端协同（记忆/技能/看板）
- 离线缓存支持
- 增量同步

## 配置

- 主配置: `~/.clawshell-local/config/config.yaml`
- 云端配置: `~/.clawshell-local/config/cloud.json`
- 日志: `~/.clawshell-local/logs/`
